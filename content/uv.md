# Install

## [[Homebrew]]

```sh
brew install uv
```

## Generic Linux

```sh
curl -LsSf https://astral.sh/uv/install.sh | sh
```

# Activate

To activate an environment, simply source the appropriate file in the [[venv]].

> [!example] Default [[venv]] with [[fish]]
> ```fish
> source .venv/bin/activate.fish
> ```
