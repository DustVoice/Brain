I currently use [[Ghostty]]. If I use it with WSL, I use it directly from within.

> [!info]
> Previously, I used [[Wezterm]], installed on Windows, to access my WSL instance(s), but as I transitioned to using [[Chezmoi]] for my dotfile management, I currently don’t have a great workflow for deploying my dotfiles on Windows.
> Therefore, I tried out using the terminal emulator directly from within WSL and grew to like it.
>
> For me, personally, this isn’t a problem, as I'm of having my complete setup in WSL.
> Managing two systems and keeping on top of everything is honestly simply too taxing.
>
> If you want to use [[Wezterm]], I would recommend using it from the Windows side, utilizing its feature of directly attaching to WSL, instead of using it from within WSL, however. I encountered multiple bugs, like the inability to maximize, for example, that made it a PITA to use from _within_ WSL.
