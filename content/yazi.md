[Yazi](https://yazi-rs.github.io) is a terminal file manager, written in [[Rust]].

I mainly use it from within [[Helix]].

## Install

### #OS/Fedora

Install it from [[Copr]].

```sh
sudo dnf copr enable lihaohong/yazi
sudo dnf install yazi
```
