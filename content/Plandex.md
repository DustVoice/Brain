## Config

[API keys and models](https://docs.plandex.ai/models/model-providers/#api-keys--environment-variables) are configured using environment variables

> [!example]
>
> ```fish "api-key"
> set -Ux OPENROUTER_API_KEY "api-key"
> ```

> [!todo] Replace
> - `api-key`: Your API key
