Scooter is an interactive Search and Replace tool for the terminal, written in [[Rust]].

## Install

### [[Cargo#binstall]]

```sh
cargo binstall scooter
```

### Helix

> [!caution] Broken
> Currently broken, at least for me.
> Issue unknown, could be an issue with the [[Helix#Steel]] fork, or with this plugin.

Scooter is available as [the `scooter.hx` plugin](https://github.com/thomasschafer/scooter.hx), for [[Helix#Steel\|Helix's plugin system]].

To install it, you can use [[Steel#Forge]]:

```sh
forge pkg install --git https://github.com/thomasschafer/scooter.hx.git
```
