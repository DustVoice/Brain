[Steel](https://github.com/mattwparas/steel) is an embedded scheme interpreter for Rust. It's most prominent use is as the  backbone for [[Helix]]'s upcoming plugin system.

## Forge

Forge is steel's package manager.

You can install most plugins using the following syntax

```sh "URL"
forge pkg install --git URL
```

> [!todo] Replace
> - `URL`: GitHub repository URL of the plugin
